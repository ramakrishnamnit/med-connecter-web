# Med Connecter Web

A modern web application for connecting patients with healthcare providers in the Netherlands.

## Getting Started

### Prerequisites

- Node.js & npm - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

### Installation

```sh
# Step 1: Clone the repository
git clone https://github.com/ramakrishnamnit/med-connecter-web.git

# Step 2: Navigate to the project directory
cd med-connecter-web

# Step 3: Install dependencies
npm install

# Step 4: Start the development server
npm run dev
```
